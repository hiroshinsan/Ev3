# Ev3 UI

Vanilla React and Tailwind CSS User Interfaces.

## Install

```bash
$ npm install @ev3/ui
```

### For Tailwind CSS

In `tailwind.config.js` add the following configuration

```js
module.exports = {
  content: [
    //...
    './node_modules/@ev3/ui/**/*.{jsx,tsx}',
    '../node_modules/@ev3/ui/**/*.{jsx,tsx}'
    //...
  ],
  //...
}
```
